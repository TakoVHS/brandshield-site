# Compliance & Data Sources

Brandshield — **defensive-only security intelligence**.

## Источники данных
- Только **публичные источники** и/или данные, предоставленные клиентом.
- Только **официальные API** или открытые интерфейсы.

## Что запрещено (non-negotiable)
- обход ToS, rate-limit circumvention
- доступ к приватным чатам/DM
- credential abuse / скрытый сбор закрытых данных
- любые offensive-инструкции (фишинг/взлом)

## Как мы формулируем выводы
- “suspected / potential impersonation”
- “signals, not accusations”
- “human review required”
