# Methodology (signals, not accusations)

## Что такое “signal”
Signal — это объяснимый индикатор риска, который требует human review.

## Explainability
- reason codes
- matched entities (official domain / keyword)
- score breakdown (similarity/confidence)

## Ограничения
- Возможны FP/FN
- Решения и внешние действия — только после проверки человеком
