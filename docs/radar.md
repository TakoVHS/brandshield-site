# Brand Abuse Radar (Weekly)

**Идея:** еженедельный публичный обзор паттернов имперсонации бренда и look‑alike доменов.  
**Правила:** только агрегаты, публичные сигналы, никакого PII, никакой “виновности” — только risk signals + human review.

## Шаблон выпуска
Скопируй структуру и публикуй 1 раз в неделю:

- Executive summary (5–7 строк)
- Data sources & posture (public only)
- Key metrics (агрегаты)
- Pattern watch (3–5 паттернов)
- Top defensive recommendations (24–72h)
- CTA

**Совет:** первый выпуск можно сделать “маленьким”, главное — регулярность.
